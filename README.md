Intro:

I recently completed an intelligent chatbot project designed to manage and automate user interactions efficiently.
This system supports dynamic add, update, delete, and fetch operations with seamless integration between the Flask 
backend and SQLiteAlchemy database.

Tech Stack

1) Python (Flask Framework)
2) AI-based chatbot development
3) SQLiteAlchemy for data handling
4) RESTful API integration & testing
   
Working

1) Auto Login System
2) When someone enters their email, the system automatically checks if it exists in the database.
3) If yes → the user gets access to the chat.
4) If not → access is denied.

on the command, it runs the CRUD operation:

1) Add: creates a new user entry
2) Delete: removes an existing user
3) Update: modifies specific fields
4) shows user details
