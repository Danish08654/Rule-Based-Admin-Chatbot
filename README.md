Step-by-Step Working

Auto Login System

When someone enters their email, the system automatically checks if it exists in the database.
If yes → the user gets access to the chat.
If not → access is denied.
 on the command, it runs the correct CRUD operation:

Add: creates a new user entry
Delete: removes an existing user
Update: modifies specific fields

View : shows user details
